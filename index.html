Simple Grid

A super simple grid built with sass. Check it out on Github.

1. Defining a Grid

Using the .container class, it gives the element its on a width of 100%, it hides the overflow, and sets the margin to auto.

2. The Columns

The columns have a certain width, have display: inline-block, and have special gutter margins.

3. Customizing the Grid

You can customize the grid super easily, just go into the stylus file, grid.styl and then change the variables. Its that simple!

4. Contact, Github & NPM
